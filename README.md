# Online Attendance System 
It is attendance system used to record students' attendance in online classes by recognising their face.

Models used are 
* MTCNN
* Facenet
* SVM Classsifier

Libraries used are:
* tensorflow
* keras
* sklearn
* matplotlib
* numpy
* PIL

For backend :
* Django 

For Frontend :
* HTMl , CSS
